## Overview
StudyHub is a Fullstack web application designed for students to organize, track, and visualize their academic progress. The system allows users to manage tasks, sync them with Google Calendar, and monitor their monthly progress through an interactive dashboard.

## Key Features
- **Task Management:** Create, edit, and delete study tasks.
- **Google Calendar Integration:** Bi-directional sync using Google Calendar API.
- **Progress Visualization:** Dynamic progress bar showing monthly completion rates.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using CSS Media Queries.
- **User Authentication:** Personal data isolation based on user login.

## Technologies Used
- **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+).
- **APIs:** Google Calendar API, OAuth 2.0.
- **Deployment:** Netlify & GitHub Actions.
- **Storage:** Web Storage API (LocalStorage).

## How to Run Locally
1. Clone the repository: `git clone https://github.com/MichalZadaStudent/StudyHab.git`
2. Open `index.html` in your browser.
